# 卜卜斋 - 周易算卦·答案之书 小程序规范

## 1. Concept & Vision

「卜卜斋」是一款以周易算卦为灵感的娱乐解压工具，以水墨国风为视觉基调，通过趣味性的铜钱投掷动画和现代MBTI风格的解读卡片，为用户提供轻松有趣的心灵解压体验。整体风格典雅而不古板，神秘而不迷信，是年轻人茶余饭后的趣味话题工具。

## 2. Design Language

### 美学方向
- 国风水彩：水墨晕染效果，渐变过渡
- 传统元素：周易卦象、铜钱、祥云、水墨山水
- 年轻化表达：MBTI趣味梗图风格解读

### 色彩系统
```
主色（墨色）: #2C3E50
次色（朱砂红）: #C0392B
强调色（金色）: #D4AF37
背景色（宣纸白）: #F5F0E6
辅助色（青墨）: #34495E
渐变背景：linear-gradient(135deg, #F5F0E6 0%, #E8DCC8 50%, #D4C4A8 100%)
```

### 字体
- 标题：思源宋体 / Noto Serif SC（备选：system-ui）
- 正文：思源黑体 / Noto Sans SC（备选：system-ui）
- 英文/数字：LXGW WenKai

### 间距系统
- 基础单位：8px
- 页面边距：24px
- 卡片间距：16px
- 组件内边距：16px/24px

### 动效哲学
- 铜钱旋转：CSS3 3D transform，800ms ease-in-out
- 卡片翻转：rotateY 180deg，600ms ease-in-out
- 页面切换：fade + slide，300ms
- 标签滚动：smooth scroll

## 3. Layout & Structure

### 页面架构（5个界面）
```
界面1：首页（Entry）
界面2：起卦页（Divination）
界面3：六十四卦图鉴（Gallery）
界面4：卦象解读卡（Card）
界面5：问题解答页（Answer）
```

### 路由逻辑
```
界面1 → 界面2 → 界面4 → 界面5 → 界面2
              ↓
           界面3 → 界面2
```

### 响应式策略
- 基准宽度：375px（小程序常用尺寸）
- 最大宽度：480px（居中显示）
- 全屏沉浸式体验

## 4. Features & Interactions

### 界面1：首页
- 居中游戏名称「卜卜斋」，毛笔字体
- 主按钮「开始算卦」，点击跳转界面2
- 底部合规声明小字

### 界面2：起卦页
- 问题输入框（最大50字）
- AI推荐问题标签（横向滚动）
- 右上角图鉴入口
- 3枚铜钱静态/动画状态
- 投掷流程：输入问题 → 点击铜钱 → 旋转动画(800ms) → 随机结果 → 1.5秒后跳转界面4

### 界面3：六十四卦图鉴
- 4×16网格布局
- 已解锁：亮色+可点击
- 未解锁：暗色+不可点击
- 按解锁状态排序（已解锁前置）
- 点击卦象可查看简介

### 界面4：卦象解读卡
- 正面：趣味年轻化解读（MBTI风格）
- 点击翻转：显示卦名+卦辞+爻辞+白话解读
- 再次点击：跳转界面5

### 界面5：问题解答页
- 左上角缩小卦象卡片
- 中间：AI安慰式短句回复（2-4句）
- 底部「返回主页」按钮

## 5. Component Inventory

### 按钮组件
- 默认：朱砂红背景，白色文字，圆角12px
- 悬停：加深10%
- 点击：缩小0.95
- 禁用：灰色，不可点击

### 铜钱组件
- 静态：金色铜钱图案，正反面随机
- 旋转中：3D旋转动画
- 落地：轻微弹跳效果

### 卦象卡片
- 正面：渐变背景+卦象符号+趣味文字
- 翻转中：3D翻转过渡
- 反面：宣纸纹理+卦辞内容

### 输入框
- 宣纸色背景，墨色边框
- 聚焦：朱砂红边框
- placeholder：水墨灰色

### 标签组件
- 圆角胶囊形状
- 默认：浅色背景
- 选中：朱砂红背景

## 6. Technical Approach

### 技术栈
- 纯HTML5 + CSS3 + Vanilla JavaScript
- 无框架依赖，单文件部署
- LocalStorage状态持久化

### 数据结构
```javascript
// 64卦数据
{
  id: number,
  name: string,
  gua: string,        // 卦象符号
  guaci: string,      // 卦辞
  yaoci: string[],    // 爻辞（6个）
  modern: string,     // 现代解读
  unlocked: boolean
}

// 投掷结果
{
  question: string,
  gua: number,        // 卦ID
  coins: number[]     // [1,0,1] 铜钱正反
}
```

### 动画规范
- 铜钱旋转：transform: rotateX(720deg)，duration: 800ms
- 卡片翻转：transform: rotateY(180deg)，duration: 600ms
- 页面切换：opacity + translateY，duration: 300ms
